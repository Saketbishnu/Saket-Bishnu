# 🚀 Saket Bishnu
A modern full-stack MERN portfolio website built using React, Node.js, Express.js, and MongoDB Atlas.

🌐 Live Website: https://saket-bishnu.vercel.app/

---

## 📌 Features

- Modern responsive UI
- Full-stack MERN architecture
- Contact form with MongoDB integration
- Dynamic API handling
- Smooth frontend experience
- Cloud deployment
- Real-time form submission handling
- Professional developer portfolio design

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- Framer Motion

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

### Deployment
- Vercel (Frontend)
- Render / Node Server
- MongoDB Atlas
