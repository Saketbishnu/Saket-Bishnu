# вҡЎ Installation Guide

## Clone Repository

```bash
git clone https://github.com/Saketbishnu/Saket-Bishnu.git
```

## Install Dependencies

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```
